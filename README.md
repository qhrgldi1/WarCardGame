# War Card Game

## Goal
1. Deal 버튼을 누르면 두 개의 카드 이미지가 랜덤으로 나타남
2. 왼쪽 카드는 Human Player, 오른쪽 카드는 CPU Player
3. 더 높은 카드가 나온 플레이어가 1점의 점수를 획득함
4. 단, 비길시 양 플레이어 모두 점수 획득 없음
5. Reset 버튼을 누르면 두 개의 카드는 뒷면으로 뒤집어지며, 점수 모두 초기화
